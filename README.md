
Segmentação de Clientes com Machine Learning e Power BI

Projeto desenvolvido em parceria com a Data Science Academy como parte dos estudos em Machine Learning e Business Intelligence.

📌 Objetivo

O objetivo deste projeto foi aplicar técnicas de Machine Learning para segmentação de clientes, utilizando o algoritmo K-Means, e posteriormente visualizar os resultados em dashboards no Power BI.

🚀 Tecnologias Utilizadas
Python
Pandas
Scikit-Learn
Jupyter Notebook
Power BI
Machine Learning (K-Means Clustering)

📊 Etapas do Projeto
Carregamento e exploração dos dados dos clientes
Análise exploratória
Padronização das variáveis
Criação do modelo de segmentação utilizando K-Means
Classificação dos clientes em clusters
Exportação dos resultados
Construção de dashboards no Power BI para análise visual dos segmentos

🧠 Conceitos Aplicados
Clusterização de dados
Segmentação de clientes
Pré-processamento de dados
Visualização de dados
Business Intelligence

📁 Estrutura do Projeto
Lab7.ipynb → Notebook principal com todo o processo de análise e modelagem
Lab7-Final.pbix → Dashboard final no Power BI
dados/ → Base de dados utilizada e arquivos gerados
📈 Resultado

O projeto permitiu identificar diferentes perfis de clientes com base em características como:

Idade
Renda anual
Pontuação de gastos

A segmentação facilita análises estratégicas e pode auxiliar empresas em ações de marketing, personalização e tomada de decisão.

🎓 Créditos

Projeto desenvolvido para fins educacionais em conjunto com a Data Science Academy, com foco em aprendizado prático de Machine Learning e Power BI.
